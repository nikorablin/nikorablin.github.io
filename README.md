# [nikorablin.io](https://nikorablin.io)

My personal website and resume.

## Built with
* [Next.js](https://nextjs.org/) App Router with static export
* [React 19](https://react.dev/)
* [TypeScript](https://www.typescriptlang.org/)
* [Tailwind CSS 4](https://tailwindcss.com/)
* [React Markdown](https://github.com/remarkjs/react-markdown) for homepage content
* [Font Awesome](https://fontawesome.com/) for icons
* [Vitest](https://vitest.dev/) and [Testing Library](https://testing-library.com/) for tests
* [semantic-release](https://semantic-release.gitbook.io/semantic-release/) for automated releases

## Development
```bash
pnpm install
pnpm dev
pnpm test
pnpm build
```
